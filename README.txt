Main functionalites:
 - Extract names of the presidents from the names of the text files
 - Associate a first name with each president
 - Display the list of president's names
 - Convert files to lowercase into a 'cleaned' folder
 - Remove the punctuation in the files stored in the 'cleaned' folder
 - Display the TF-Score of each word for each file
 - Display the IDF-Score for each word
 - Display the TF-IDF Matrix

Instructions:
Execute the main.py program and everything should work.
